# Reverie Coaster Club

Sign-up SPA for the Reverie Brewing Co. Coaster Club.  
Members scan a QR code, enter their details, and submissions land directly in a Google Sheet.

---

## Repo Structure

```
reverie-coaster-club/
├── assets/
│   ├── rbco-logo.jpg               # RBco. circular logo
│   └── Rev-Coaster-Club_v11.stl    # Coaster STL (used to render preview PNG)
├── src/
│   ├── index.template.html         # HTML template (base64 blobs are placeholders)
│   └── Code.gs                     # Google Apps Script backend
├── dist/                           # Built output (git-ignored, generated by build.py)
│   ├── index.html                  # Final self-contained HTML
│   └── coaster_preview.png         # Rendered coaster top-down PNG
├── .github/
│   └── workflows/
│       └── deploy.yml              # CI: build → GitHub Pages + Apps Script
├── .clasp.json                     # clasp config (add your scriptId)
├── build.py                        # Build script
└── README.md
```

## Local Development

### Prerequisites

```bash
pip install numpy pillow
npm install -g @google/clasp
```

### Build

```bash
python build.py
```

Outputs `dist/index.html` and `dist/coaster_preview.png`.

### Build + Serve

```bash
python build.py --serve
# → http://localhost:8080
```

### Build + Watch for changes

```bash
pip install watchdog
python build.py --watch
```

## First-Time Setup

### 1. Google Sheet

1. Go to [sheets.google.com](https://sheets.google.com)
2. Create a new spreadsheet, name it **Reverie Coaster Club**
3. Copy the Sheet ID from the URL:  
   `https://docs.google.com/spreadsheets/d/`**`SHEET_ID`**`/edit`
4. Open `src/Code.gs` and replace `YOUR_SPREADSHEET_ID_HERE` with your Sheet ID

### 2. Apps Script

1. Go to [script.google.com](https://script.google.com) → New Project
2. Paste in the contents of `src/Code.gs`
3. **Deploy → New Deployment**
   - Type: **Web App**
   - Execute as: **Me**
   - Who has access: **Anyone**
4. Copy the **Web App URL**
5. Open `src/index.template.html`, find this line and replace the URL:
   ```js
   const APPS_SCRIPT_URL = "YOUR_APPS_SCRIPT_WEB_APP_URL_HERE";
   ```
6. Copy the **Script ID** from the Apps Script URL:  
   `https://script.google.com/home/projects/`**`SCRIPT_ID`**`/edit`
7. Paste it into `.clasp.json`:
   ```json
   { "scriptId": "YOUR_SCRIPT_ID_HERE", "rootDir": "./src" }
   ```

### 3. GitHub Repository

1. Create a new repo on GitHub (e.g. `reverie-coaster-club`)
2. Push this project:
   ```bash
   git init
   git add .
   git commit -m "Initial commit"
   git remote add origin https://github.com/YOUR_USERNAME/reverie-coaster-club.git
   git push -u origin main
   ```
3. Go to **Settings → Pages** → Source: **gh-pages branch** → Save  
   Your site will be live at `https://YOUR_USERNAME.github.io/reverie-coaster-club`

### 4. GitHub Secrets

Go to **Settings → Secrets and variables → Actions** and add:

| Secret | Value |
|---|---|
| `CLASP_CREDENTIALS` | Contents of `~/.clasprc.json` after running `clasp login` |
| `DEPLOYMENT_ID` | From Apps Script → Deploy → Manage Deployments |

After that, every push to `main` will:
- Rebuild `dist/index.html`
- Deploy to GitHub Pages
- Push `Code.gs` to Apps Script and redeploy

### 5. QR Code

Generate a QR code pointing to your GitHub Pages URL using any free tool  
(e.g. [qr-code-generator.com](https://qr-code-generator.com)) and print it for the bar.

## Migrating to Reverie's Google Account

When you're ready to hand off:

1. Copy the Google Sheet to their Drive
2. Repeat Steps 1–2 above under their account
3. Update `APPS_SCRIPT_URL` in `src/index.template.html` with the new Web App URL
4. Update `.clasp.json` with the new Script ID
5. Update GitHub Secrets with new `CLASP_CREDENTIALS` and `DEPLOYMENT_ID`
6. Push to `main` — the Action handles the rest

---

## Updating the Coaster STL

1. Drop the new `.stl` file into `assets/`
2. Update the `STL_PATH` in `build.py` if the filename changed
3. Push — the Action re-renders the PNG and rebuilds automatically

## Year Rollover

The Google Sheet creates a new tab automatically each year (e.g. `2026`, `2027`).  
No action needed.
